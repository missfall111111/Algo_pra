1. 关于vector的越界，如果你[]访问的元素索引大于vector中存储的元素的最大索引，就会越界。区分map的[]操作，当第一次访问时，他会自动创建一个对象。

```cpp
int main() {
	vector<int>a;
	a[0] = 1;  //错误！直接越界了！
	cout << a[0];
	return 0;
}
```
